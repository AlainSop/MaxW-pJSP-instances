The first main parameter of our algorithm to generate MaxW instances is the global desired off shift density $\mathcal{D}$. It expresses the minimal overall proportions of rest days that each operator should have over the entire horizon ($\mathcal{H}$). These rest days are distributed across multiple time intervals, which are randomly chosen but must respect some constraints. Each interval should be at least 3 days, and at most $0.2*\mathcal{H}$ days (neither too short nor too long). Within each interval, the number of rest days is related to the other main parameter, the local minimal density ($d$). They express the overall min proportion of rest days that the operator should have over any specific MaxW interval. These proportions are chosen with random gaussian probability: $\mathcal{N}(d, d/4)$. This method introduces some unpredictability into when rest periods occur and how long they are.
